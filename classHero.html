◆ levelUp()

void Hero::levelUp ( )
protected

When a units XP is above or equal to 100, it is called.

If a unit has N times 100 XP it is called N times. Sets a units maximumHP and ATK to 1.1 times the current value. Also it sets current HP to maximumHP. lvlUp() method is used by in the deliverHit() method.
